> 🎲 *언더테일의 도덕적 전투 선택을 다이스 포커 메커니즘으로 구현한 1인 개발 로그라이크.*
> 모든 조우는 하나의 선택: **싸우기(Fight)** 또는 **대화하기(Talk)**. 같은 주사위, 같은 패 — 하지만 다른 의미로 분기합니다.

<details>
<summary><b>⚔️ 핵심 시스템 (148개 스크립트, 약 22,000 LOC)</b></summary>
<br>

- **PedigreeManager** — 10단계 주사위 패 순위(HighCard → MonoRoll) + 이중 피해/공감력 계산
- **TalkManager** — 공감 게이지, 적별 조건 퍼즐(7가지 스위치 타입), 대화 분기
- **BattleManager SubSM** — 11상태 턴제 전투 FSM(Intro → RouteSelect → Dice → Hand → Resolve → TurnEnd)
- **DiceSlotManager** — 5슬롯 장착/교체/인벤토리 시스템(기본/고유/강화 3티어 주사위)
- **SoulGimmick System** — ISoulGimmickHandler 인터페이스 + 팩토리 패턴을 통한 10가지 고유 영혼 기믹(Bell, Ember, Gavel, Hollow, Palette, Phonograph, Roulette, Scribe, Seven, TheDealer)
- **CorridorState** — 22노드 고정 복도 진행(전투/정예/보스/휴식/이벤트/출구)
- **Fight/Talk 대칭 루트** — 동일한 다이스-핸드 루프가 공격과 공감 결과로 분기
- **EnemyWeakness System** — 시각적 배지 + 퍼즐 기반 약점 힌트 + 동적 편향 메커니즘
- **MemoryFragment** — FragmentEffectHelper를 갖춘 소모품 런 간 아이템 시스템
- **EmotionalStateTracker** → BattleAtmosphereController + DynamicUITintController + EnemyExpressionController
- **FlashbackManager** — 마이크로/단편/장편 3티어 플래시백 시네마틱 시퀀스
- **CrossRunSaveManager** — 메타 이벤트 추적 기능을 갖춘 JSON 기반 영속성
- **MetaEventManager** — 런별 타이틀 로고/서브타이틀 변경 + 딜러 타이핑 방향
- **트루 엔딩 루트** — Fight 및 Talk 엔딩 모두 달성 시 해금
- **LocalizationManager** — 정적 UI 번역을 포함한 KO/EN/JA/RU 4개국어 지원
- **AccessibilityManager** — 모션 감소, 색맹 오버레이, 시각적 사운드 표시기, 글꼴 크기 조절

</details>

> 하나의 바이너리. 제로 의존성. Unity Editor로의 직접 HTTP 브리지.

AI 에이전트와 터미널에서 Unity Editor를 직접 제어하는 오픈소스 CLI 도구입니다. 단일 Go 바이너리가 localhost HTTP로 Unity와 통신하며, Unity 측 커넥터는 에디터 실행 시 자동 시작합니다.

- **Go CLI** — 20개 파일, 약 800 LOC — 명령어 파싱, HTTP 클라이언트, 인스턴스 디스커버리
- **C# Connector** — 22개 파일, 약 2,300 LOC — `[UnityCliTool]` 어트리뷰트 기반 자동 도구 등록
- **핵심 명령어** — `editor play`, `test`, `status`, `exec`, `console`, `update`
- **아키텍처** — Stateless 단발 HTTP, zero runtime dependencies, cross-platform (Linux/macOS/Windows)
